# Task 1: Data Immersion & Wrangling

## Overview
This project focuses on data acquisition, data quality assessment, data cleaning, transformation, and preparation of a dataset for analysis. The objective was to identify data quality issues, perform preprocessing, and create an analysis-ready dataset.

---

## Project Objectives
- Understand the dataset structure and variables.
- Assess data quality issues such as missing values and duplicates.
- Perform data cleaning and preprocessing.
- Create new features for better analysis.
- Generate visualizations for data insights.
- Produce a cleaned dataset ready for further analysis.

---

## Dataset Information
The dataset contains sales-related information including customer details, product information, order details, and sales transactions.

---

## Tasks Performed

### 1. Data Access & Familiarization
- Loaded the dataset using Pandas.
- Explored dataset structure and columns.
- Reviewed data types and dataset dimensions.
- Created a Data Dictionary describing dataset attributes.

### 2. Data Quality Assessment
- Identified missing values.
- Checked for duplicate records.
- Examined dataset consistency.

### 3. Data Cleaning & Transformation
- Removed duplicate entries.
- Handled data quality issues.
- Created a new feature:
  - `TOTAL_VALUE = SALES × QUANTITYORDERED`
- Prepared a clean dataset for analysis.

### 4. Data Visualization
- Generated a Sales Distribution Histogram.
- Saved visualization as an image file.

### 5. Output Generation
- Exported cleaned dataset to CSV format.
- Organized project files for reproducibility.
